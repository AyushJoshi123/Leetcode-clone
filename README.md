# LeetCode Clone

A simple clone of LeetCode for practicing algorithm problems, tracking progress, and solving challenges in a clean web interface.

## Features

- Problem list with categories and difficulty
- Code editor with syntax highlighting
- Submission and result feedback
- User authentication and progress tracking
- Search and filter problems

## Tech Stack

- Frontend: React, TypeScript, CSS
- Backend: Node.js, Express
- Database: PostgreSQL / MongoDB
- Editor: Monaco or CodeMirror

## Installation

1. Clone the repo
    ```bash
    git clone https://github.com/your-username/leetcode-clone.git
    cd leetcode-clone
    ```
2. Install dependencies
    ```bash
    npm install
    ```
3. Configure environment variables
    - `DATABASE_URL`
    - `JWT_SECRET`
    - `PORT`

4. Start the app
    ```bash
    npm run dev
    ```

## Usage

- Open `http://localhost:3000`
- Sign up or log in
- Browse problems, write code, submit solutions
